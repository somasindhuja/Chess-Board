# ♟️ Chess Board Project

This is a simple **Chess Board UI** built using **HTML and CSS only**.

---

## 🚀 Features
- 8x8 chess board layout
- Proper black and white pattern
- Chess pieces using image URLs
- Clean and simple design
- No JavaScript used

---

## 🛠️ Technologies Used

- CSS3
